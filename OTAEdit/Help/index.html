OTA Edit is a modding utility meant to make it easier to modify Total Annihilation's .ota file format. These files are used by TA's maps to specify general map settings, as well as more specific parameters used for campaigns.

These help pages will guide you through what each field does, and what data is expected to be inputted.

Thanks to the author (or authors) of this OTA File Format page. OTA Edit would have taken a lot longer to develop without the info on that page.
