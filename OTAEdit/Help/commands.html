Commands

The initial mission field for schema added units is essentially where the bulk of mission making happens.

Command Description
m X, Y Move to coordinates X and Y
p X, Y Patrol between the unit's starting location and the coords given in the command.
a UNITNAME Attack the nearest specified unit type. UNITNAME can be the internal name for a unit type, or it can be a specific unit's identifier.
